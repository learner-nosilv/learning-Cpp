## 실습문제 리뷰

> **12**  
> `어려웠던 것 1` **O(n*n)이 아닌 Sort를 구현하는게 배열에서 난해했음**  
> `어려웠던 것 2` ** new delete 잊지않기**  
   
   학교 자료구조 수업에서 Sorting 을 주제로 배웠었는데  
   배열로 구현하려니 오히려 더 복잡해졌다.  
  
   기왕이면 메모리 누수도 안 나게해야지 하면서  
   new-delete를 체크하는 것에도 시간과 노력이 들었다.
   동적할당 개념이 없는 언어를 개발한 이유...! 납득-
